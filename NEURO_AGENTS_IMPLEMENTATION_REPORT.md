## Модель данных

Источник: `src/data/neuroAgentsConfig.js`

- **`NEURO_AGENTS`** — библиотека и хиро-мини-карточки (поля `name`, `category`, `description`, `task`, `status`, `statusKind`, `iconType`, `accentColor`, …).  
- **`NEURO_DEMO_ITEMS`** — ряд из 5 демо (связь с агентом через `agentId`, `scenario`, `duration`, `thumbnailType`, `videoUrl`).  
- **`NEURO_WHY_US`** — 4 карточки «почему мы».  
- **`NEURO_IMPACT_KPIS`**, **`NEURO_CHART_SERIES`**, **`NEURO_ROI_DEFAULTS`** — блок impact/ROI.

### Как добавить нового нейроагента

1. Добавьте объект в массив **`NEURO_AGENTS`** (уникальный `id`, тексты, `statusKind`, `iconType`, `accentColor` из пресетов: `ember` \| `violet` \| `ice` \| `copper` \| `teal`).  
2. При необходимости добавьте ветку в **`NeuroAgentHubIcon`** (`NeuroAgentsIcons.jsx`), если новый `iconType`.  
3. Для превью в библиотеке цвет задаётся классом `na-lib-card--{accentColor}` в CSS (при новом пресете — добавить блок `.na-lib-card--…` и `.na-hero__mini--…`).

### Как добавить демо / видео позже

1. Добавьте элемент в **`NEURO_DEMO_ITEMS`** или расширьте существующий: `videoUrl`, при необходимости `thumbnailUrl` (потребуется доработка компонента под картинку).  
2. В `NeuroAgentsDemoGallery.jsx` кнопка play уже отключается без `videoUrl`; после появления URL подключите обработчик (модалка / новое окно).

## Оставшиеся риски / зазоры

- Пиксельная 1:1 с PNG без макетных ассетов (3D-головы в библиотеке) невозможна: визуалы собраны на CSS-градиентах/силуэтах.  
- ROI остаётся **демонстрационной формулой**, не финмоделью.  
- Browser MCP в данной сессии не поднял вкладку — регрессию UI выполнить локально.
