# SITE ARCHITECTURE REVIEW V2

## Какие замечания учтены
- Усилена системная навигация и смысл первого экрана: что такое AKMEHR, как работает, кому подходит.
- Разделение B2C/B2B/B2G сохранено и усилено по сценариям принятия решения.
- Добавлены коммерческие и воронко-ориентированные блоки для B2B.
- Усилен блок доверия: контакты, документы, политика данных, партнерский контур.
- Интеграции показаны как технологическая зрелость, а не витрина.
- Нейро-сотрудники переведены в формат продуктовой витрины с демо-статусами.

## Что меняется по разделам
- `HeroSection`: объяснение ценности платформы, схема "как работает", сегменты "кому подходит", обновленный AI-текст.
- `B2CSection`: разделение на 3 уровня (диагностика -> результат -> развитие), два набора переключателей.
- `B2BSection`: добавлен блок "что можно купить/заказать", панель "до/после", схема бизнес-контура.
- `B2GSection`: усилен формат программ и блок "что получает организация".
- `NeuroAgentsSection`: витрина модулей с карточками и детальным профилем выбранного модуля.
- `IntegrationsSection`: добавлены пояснение зрелости и отдельные контуры передачи данных в amoCRM/BS-Evolve.
- `SiteConceptSection`: добавлен блок контактов и доверия, расширена карта сайта под целевую архитектуру.
