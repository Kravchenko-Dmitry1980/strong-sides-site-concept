## 1. Орбита «колесо обозрения»

### Проблема

Внешний контейнер орбиты вращался (`rotate` на `.na-hero__orbit-rotator`), а на карточках оставалось только CSS-контрвращение `-60° × slot`, **без** учёта угла контейнера θ. В итоге карточки визуально «крутились» вместе с орбитой, подписи уходили под диагональ.

### Решение

- Вынесен компонент **`HeroOrbitMini`** (`NeuroAgentsHero.jsx`).
- Для каждой карточки: `useTransform(orbitRotation, (theta) => -(theta + index * 60))` → передаётся в `motion.div` как **`style={{ rotate: uprightDeg }}`**.
- Логика: слот по-прежнему ставит карточку на окружность через `rotate(i·60°) translateY(-R)` в CSS; суммарный поворот карточки в мире = **θ + i·60°**; контрвращение **−(θ + i·60°)** держит содержимое **горизонтальным** (как колесо обозрения).

### Скорость

- Автовращение: было **0.035°/кадр** → стало **0.009°/кадр** (~в 4 раза медленнее, спокойнее).
- Drag: множитель **0.42** → **0.22** (мягче, без резких рывков).
- При `prefers-reduced-motion: reduce` автокрут по-прежнему отключается.

### CSS

- У `.na-hero__mini` **удалён** фиксированный `transform: rotate(calc(var(--na-slot) * -60deg))` — угол задаёт только Framer.

## 2. Библиотека: зона портрета

### Отдельные PNG

В репозитории **нет** каталога `references/neuro-agents/agents/*.png` (проверены только `01`…`06` в `references/neuro-agents/`).

### Реализация

- В **`neuroAgentsConfig.js`** у каждого агента добавлено поле **`portraitSrc: null`** — при размещении файлов в `public/` (например `/neuro-agents/agents/neuro-sales.png`) достаточно прописать URL: карточка отрисует **`<img>`**.
- Новый модуль **`NeuroAgentLibraryPortrait.jsx`**: для каждого `id` — **отдельный SVG-портрет** (силуэт «нейро-головы», разные акценты: мишень/продажи, график/аналитика, гарнитура/секретарь, HR, трекер с ростом, KPI/donut). Уникальные `id` у градиентов в `<defs>`, чтобы не конфликтовать при 6 карточках на странице.
- **`NeuroAgentLibraryCard.jsx`**: обёртка `.na-lib-card__portrait-wrap` + портрет-компонент.
- **`neuro-agents.css`**: старые CSS-only «плашки» `.na-lib-card__portrait--*` удалены; добавлены стили для **`.na-lib-card__portrait-svg`** и **`.na-lib-card__portrait-img`**.
