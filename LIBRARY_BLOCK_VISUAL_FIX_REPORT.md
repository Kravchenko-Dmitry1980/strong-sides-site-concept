## Медиазона карточки (что заменило «слабые» плейсхолдеры)

- Единый **силуэт головы в профиль** (`HEAD_PATH`) — масштаб и композиция ближе к референсу, не овал-яйцо с иконкой.
- **Радиальное ядро** с цветом роли (оранж/золото, фиолет, ледяной синий, янтарь HR, бирюза, фиолет KPI).
- **Слой частиц**: паттерн + набор точек внутри `clipPath`, режим `screen` — эффект «светящихся данных».
- **Контур** с лёгким `feGaussianBlur` + тонкая обводка для премиального кольца.
- **Виньетка** по краям медиаблока.
- **Ролевые оверлеи** (в том же клипе, `mix-blend-mode: screen`):
  - продажник — линии «схемы» + узлы;
  - аналитик — голографические столбцы диаграммы;
  - секретарь — дуга оголовья + чаши наушников;
  - HR — сеть узлов;
  - трекер — рамка HUD + линия графика;
  - KPI — мишень в зоне «взгляда».

Поддержка **`portraitSrc`** сохранена: при выставлении URL в конфиге подставляется реальное изображение без изменения разметки карточки.
